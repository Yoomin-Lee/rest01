## 3. 디자인 품질 평가

### 3-1. 일관성

- **컬러 일관성**: palette.css의 CSS 변수 전역 적용으로 색상 일관성 보장
- **타이포그래피**: Noto Sans KR 단일 폰트 패밀리, 크기 체계 정돈
- **간격 체계**: 섹션 padding 96px, 컴포넌트 gap 통일

### 3-2. 사용자 경험 (UX)

| 요소 | 평가 | 상세 |
|------|------|------|
| 첫인상 | ✅ 강함 | Hero 타이핑 애니메이션 + 프로필 플로팅 |
| 탐색 편의성 | ✅ 양호 | 고정 Nav + 스크롤 섹션 하이라이트 |
| 인터랙션 피드백 | ✅ 있음 | 호버 효과, 커서 변화, 스파클 트레일 |
| 정보 위계 | ✅ 명확 | 섹션 타이틀 → 서브타이틀 → 본문 |

### 3-3. 독창성

- 커스텀 커서(화살표)와 파비콘 일치 → 브랜드 일관성
- 스파클 트레일 → 신비로운 분위기 연출
- 옐로우×바이올렛 팔레트 → 학술+창의 이미지 조화

## 6. 결론

본 프로젝트는 역사학 석사 출신의 주니어 웹 개발자 이유민의 포트폴리오 웹사이트로,  
실제 이력서 데이터를 정확히 반영하고 Yellow × Violet 색상 테마를 일관되게 적용하였습니다.

외부 프레임워크 없이 순수 HTML/CSS/JavaScript만으로 구현하여 학습 과정의 진정성을 보여주며,  
GitHub Pages를 통한 무료 배포와 카카오톡 공유 최적화까지 실용적인 요소를 갖추었습니다.

향후 프로젝트 섹션에 실제 작업물을 추가하고, 이미지 최적화를 진행하면  
취업 활동에 충분히 활용 가능한 완성도 높은 포트폴리오가 될 것입니다.
